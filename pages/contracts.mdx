## 🏗️ Contract: `pay()` Logic

```solidity
function pay(
    address token,
    address merchant,
    uint256 amount,
    bytes32 orderId
) public {
    // 1% to Treasury
    uint256 fee = (amount * 100) / 10000;
    // 99% to Merchant
    uint256 merchantAmount = amount - fee;

    // Direct atomic transfers
    IERC20(token).safeTransferFrom(msg.sender, protocolTreasury, fee);
    IERC20(token).safeTransferFrom(msg.sender, merchant, merchantAmount);

    emit PaymentExecuted(msg.sender, merchant, token, amount, orderId);
}
```

The `pay()` function is the engine of the protocol. It ensures atomic settlement with zero risk of funds being stuck in the contract.

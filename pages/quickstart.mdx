# ⚡ Quickstart — 1 Minute to Your First Agent Payment

Get an AI Agent to autonomously pay $0.10 USDC for a premium API request on **Base Sepolia**.

## 1. Install

```bash
npm install @paynodelabs/sdk-js
```

## 2. Agent (Payer)

```javascript
const { PayNodeAgentClient } = require('@paynodelabs/sdk-js');

async function main() {
  // Now only needs Private Key! (Defaults to Base Mainnet RPCs)
  // For Sandbox: new PayNodeAgentClient(key, 'https://sepolia.base.org')
  const agent = new PayNodeAgentClient(process.env.PRIVATE_KEY);
  
  // Requests a protected route — 402 payment is handled autonomously
  const res = await agent.requestGate('http://localhost:3000/api/data');
  const { data } = await res.json();
  console.log('✅ Received:', data);
}

main();
```

## 3. Merchant (Receiver)

Protect your API route. Defaults to **USDC on Base Mainnet**.

```javascript
const { x402_gate, PAYNODE_ROUTER_ADDRESS_SANDBOX, BASE_USDC_ADDRESS_SANDBOX } = require('@paynodelabs/sdk-js');

// 1. Minimum config for Sandbox (Base Sepolia)
app.get('/api/data', x402_gate({
  merchantAddress: '0xYOUR_WALLET',
  price: '0.10',
  chainId: 84532, 
  contractAddress: PAYNODE_ROUTER_ADDRESS_SANDBOX,
  tokenAddress: BASE_USDC_ADDRESS_SANDBOX,
}), (req, res) => {
  res.json({ data: 'Hello from AI Economy!' });
});

// 2. Minimum config for Mainnet (Base)
// app.get('/api/data', x402_gate({ merchantAddress: '0x...', price: '1.00' }));
```

### 📏 Standard Precision & Limits

PayNode uses the token's smallest unit (USDC = 6 decimals).

| Amount (USDC) | PayNode Unit | Protocol Status |
| :--- | :--- | :--- |
| **0.001 USDC** | **1,000** | **✅ Protocol Floor** |
| 1.00 USDC | 1,000,000 | ✅ Active |
| < 0.001 USDC | < 1,000 | ❌ Rejected (Dust Protection) |

# Python SDK (`paynode-sdk-python`)

The Python SDK is optimized for Python-based AI workflows (LangChain, AutoGen, CrewAI) and modern backends (FastAPI).

## 📦 Installation

```bash
pip install paynode-sdk-python web3
# Optional: for webhook delivery
pip install aiohttp
```

## 🤖 For AI Agents (Client)

> **⚠️ Prerequisites**
>
> The Agent's wallet must hold **Base ETH** (gas) and **USDC** (payment).
> First time? See [Contracts](/contracts) for testnet faucet instructions.

The `PayNodeAgentClient` intercepts `402` responses, executes on-chain payment, and retries automatically.

```python
import os
from paynode_sdk import PayNodeAgentClient

agent = PayNodeAgentClient(
    rpc_urls=["https://mainnet.base.org"],
    private_key=os.getenv("AGENT_PRIVATE_KEY")
)

# Standard requests — 402 payment is handled autonomously
response = agent.get("https://api.merchant.com/premium-data")
print("✅ Data:", response.json())
```

### Security: Private Key Protection

The SDK converts the private key to an `Account` object on initialization and discards the raw string. If a transaction fails, the error message is intercepted and wrapped in a `PayNodeException` — preventing accidental key leaks in stack traces (e.g. LangChain debug logs).

### EIP-2612 Permit (Single-Tx Payment)

For gasless single-transaction payments using an offline signature:

```python
# Payer signs offline, Agent relays on-chain
tx_hash = agent.pay_with_permit(
    router_addr="0x92e20164FC457a2aC35f53D06268168e6352b200",
    payer_address="0xPayerWallet...",
    token_addr="0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913",
    merchant_addr="0xMerchantWallet...",
    amount=1000000,         # 1 USDC (6 decimals)
    order_id="order_123",
    deadline=1716000000,    # Unix timestamp
    v=28, r=b'\x...', s=b'\x...'
)
```

## 🏪 For Merchants (FastAPI Middleware)

`PayNodeMiddleware` integrates into FastAPI as ASGI middleware.

```python
from fastapi import FastAPI
from paynode_sdk import PayNodeMiddleware
import os

app = FastAPI()

# Add PayNode payment gate as middleware
app.add_middleware(
    PayNodeMiddleware,
    rpc_url=os.getenv("BASE_RPC_URL", "https://mainnet.base.org"),
    contract_address="0x92e20164FC457a2aC35f53D06268168e6352b200",
    merchant_address="0xYourWallet...",
    chain_id=8453,
    currency="USDC",
    token_address="0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913",
    price="0.50",
    decimals=6,
)

@app.get("/api/inference")
def inference():
    # This ONLY runs after verified on-chain payment
    return {"result": "AI inference output", "paid": True}
```

### 402 Response Headers

When a request lacks `x-paynode-receipt`, the middleware returns `402` with headers identical to the [JS SDK middleware headers](/sdk-js#middleware-headers).

## 🛡️ Token Whitelist

The Verifier includes a built-in `ACCEPTED_TOKENS` whitelist per chain to block fake-token attacks.

```python
from paynode_sdk import PayNodeVerifier
from paynode_sdk.constants import ACCEPTED_TOKENS

# Auto-detect from chainId (recommended)
verifier = PayNodeVerifier(
    rpc_url="https://mainnet.base.org",
    contract_address="0x92e2...",
    chain_id=8453,
    # Automatically uses ACCEPTED_TOKENS[8453] = [USDC]
)

# Override with custom list
verifier = PayNodeVerifier(
    rpc_url="https://mainnet.base.org",
    contract_address="0x92e2...",
    chain_id=8453,
    accepted_tokens=["0xCustomToken..."],
)

# Disable whitelist (testing only)
verifier = PayNodeVerifier(
    rpc_url="http://localhost:8545",
    contract_address="0x1234...",
    chain_id=8453,
    accepted_tokens=[]  # Empty = no whitelist check
)
```

## 🔒 Thread Safety & Concurrency

The Python SDK uses `threading.Lock()` for atomic nonce management. When multiple tools/threads send concurrent transactions, nonces are queued and incremented safely — avoiding `nonce too low` errors.

```python
# Safe for multi-threaded usage
import threading

def tool_call(agent, url):
    response = agent.get(url)  # Thread-safe nonce handling
    return response.json()

threads = [
    threading.Thread(target=tool_call, args=(agent, url))
    for url in paid_api_urls
]
for t in threads:
    t.start()
```

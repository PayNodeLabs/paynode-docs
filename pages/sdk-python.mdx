# SDK - Python (paynode-sdk-python)

Built on **Web3.py**, the Python SDK is designed for AI agents (LangChain, AutoGPT, etc.) that need to consume paid resources autonomously on Base L2.

## 🚀 Installation

```bash
pip install paynode-sdk-python
```

## 🛠️ Usage Example

```python
from paynode_sdk.client import PayNodeAgentClient

client = PayNodeAgentClient(
    rpc_urls=["https://mainnet.base.org"],
    private_key="YOUR_PRIVATE_KEY"
)

# This call handles the entire x402 handshake autonomously
response = client.post(
    url="https://api.yourmerchant.com/api/v1/protected",
    json={"agent_name": "Python-Agent-Pioneer"}
)

if response.status_code == 200:
    print("✅ Resource Accessed:", response.json())
```

## 🔐 ERC20 Allowance Management

The Python SDK automatically manages your USDC allowance. If it detects a low allowance for the PayNode Router, it will:
1. **Grant Infinite Approval** once to save gas on future transactions.
2. **Handle Nonces** by checking the pending state to avoid conflicts in high-frequency payment streams.

## ⛽ Gas Strategy

The SDK automatically boosts the current network gas price by **20%** during payment execution to ensure reliable inclusion in Base Mainnet blocks during periods of network congestion.
